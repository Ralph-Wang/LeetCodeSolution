# 题面

计算一个无符号整数二进制表示中 1 的个数


# 思路
1. 遍历所有位, 一个一个数
P.S. Python 中`位运算`并不比除/模快 =,=


1. 分治法. 相邻位相加 -> 相邻2位相加 -> 相邻 4 位相加 -> 相邻 8 位相加 -> 相邻16 位相加


以全长 8 位为例:

| 0 | 1 | 1 | 0 | 1 | 1 | 0 | 1 | 

|  01   |  01   |  10   |   01  | 

|     0010      |     0011      | 

|           0000101             |  -> 5
